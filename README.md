# :page_with_curl:프로젝트 소개

<br>
<br>

캠핑을 즐기는 이들을 위한 통합 캠핑 예약 서비스로, 다양한 캠핑 관련 정보와 서비스를 제공합니다. 
<br>
사용자들은 캠핑장 예약뿐만 아니라, 리뷰와 의견을 교환하는 커뮤니티 기능을 통해 소통할 수 있으며 캠핑에 대한 다양한 정보를 보다 편리하게 얻을 수 있습니다.
